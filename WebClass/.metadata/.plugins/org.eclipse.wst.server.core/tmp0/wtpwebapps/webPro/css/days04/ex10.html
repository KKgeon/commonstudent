[CSS 2D 변환]

    - 2D변환 ? 요소 이동, 회전, 비율 조정, 기울기
    - transform(== 변환) 속성
    1) 이동 : translate()
    2) 회전 : rotate()
    3) 비율 조정 : scale(), scaleX(), scaleY() 
    4) 기울기 : skew(), skewX(), skewY()
    약식 -> matrix()
