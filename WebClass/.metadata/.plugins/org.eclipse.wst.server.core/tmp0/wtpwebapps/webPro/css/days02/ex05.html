[반응형 테이블]
화면이 너무 작아서 전체 내용(content)을 표시할 수 없는 경우 가로 스크롤 막대로 표시하는 테이블

-table 요소 주위에 컨테이너(div)를 추가하고
overflow-x:auto 속성을 설정하면
반응형 테이블을 만들 수 있다.
